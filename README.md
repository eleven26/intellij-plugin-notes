## 内置图标使用

所有内置图标所在路径：

```
ideaIC-2019.1/lib/icons.jar
```

这里的 `ideaIC-2019.1` 是 插件项目里面的 build.gradle 里面的 intellij.version 版本的 idea

解压这个 icons.jar 即可看到哪些图标可以直接使用。
